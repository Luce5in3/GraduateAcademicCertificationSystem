# UI/UX Pro Max

## 功能描述

该技能用于重构前端项目的 UI/UX，主要包括以下功能：

1. **登录界面重构**：不使用竖屏手机界面，采用更适合桌面端的布局
2. **整体视觉效果优化**：提升界面美观度，确保风格简约
3. **内容显示优化**：合理分布内容，提高信息呈现的清晰度
4. **交互反馈优化**：添加合理的画面反馈，提升用户体验

## 适用场景

- 前端项目视觉效果不佳，需要整体优化
- 登录界面使用了竖屏手机界面，需要改为更适合桌面端的布局
- 毕业证明管理系统需要提升用户体验
- 界面风格需要调整为简约风格

## 执行步骤

1. **分析现有代码**：了解当前项目的结构和实现
2. **重构登录界面**：移除竖屏手机界面，改为更适合桌面端的布局
3. **优化其他界面**：提升整体视觉效果，确保风格简约
4. **调整内容分布**：确保内容显示合理，信息呈现清晰
5. **添加交互反馈**：为用户操作添加合理的画面反馈
6. **测试验证**：确保所有界面功能正常，视觉效果良好

## 设计原则

- **风格简约**：避免过多装饰，保持界面简洁明了
- **内容合理**：信息分布合理，易于阅读和理解
- **反馈及时**：为用户操作提供及时、合理的视觉反馈
- **响应式设计**：确保在不同设备上都能良好显示
- **一致性**：保持界面风格和交互方式的一致性

## 技术要求

- 熟悉前端开发技术栈
- 了解现代 UI/UX 设计原则
- 能够独立完成界面重构和优化工作
- 确保代码质量和性能

## 交付标准

- 登录界面不再使用竖屏手机界面
- 整体视觉效果得到明显提升
- 内容显示分布合理
- 交互反馈及时、合理
- 代码结构清晰，易于维护
