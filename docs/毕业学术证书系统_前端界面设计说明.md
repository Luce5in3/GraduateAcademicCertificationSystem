# 毕业学术证书系统（Web端）前端界面设计说明

## 一、整体设计原则

- **风格**：简约 / 清晰 / 偏政务系统风格
- **布局**：经典 Web 管理系统布局（顶部栏 + 左侧菜单 + 内容区）
- **色彩**：
  - 主色：#1677FF（蓝色，可信赖）
  - 辅色：#F5F7FA（背景灰）
  - 状态色：成功绿 / 警告橙 / 错误红
- **适配**：PC Web 为主（≥1280px），可兼容平板
- **交互原则**：
  - 表单校验即时反馈
  - 状态清晰（审批状态、进度）
  - 操作尽量不超过 3 步

---

## 二、整体页面结构

```text
┌──────────────────────────────────────────┐
│ 顶部导航栏（系统名 + 用户信息 + 退出） │
├───────────────┬──────────────────────────┤
│ 左侧功能菜单  │ 主内容区域               │
│               │                          │
│               │                          │
└───────────────┴──────────────────────────┘
```

### 顶部导航栏（Header）
- 系统名称：**毕业学术证书管理系统**
- 当前用户：头像 + 姓名 + 身份（学生 / 教师）
- 操作：
  - 个人信息
  - 退出登录

### 左侧菜单（Sider）
根据用户角色动态显示：

**学生菜单**：
- 首页
- 证明申请
- 我的申请
- 个人信息

**教师菜单**：
- 首页
- 待审批申请
- 已审批记录
- 个人信息

## 三、登录 / 注册界面

### 1. 登录界面

**布局结构**：
```text
┌───────────────────────────┐
│        系统名称           │
│  ┌───────────────────┐  │
│  │ 用户名             │  │
│  │ 密码               │  │
│  │ [ 登录 ]           │  │
│  │ 去注册             │  │
│  └───────────────────┘  │
└───────────────────────────┘
```

**字段**：
- 用户名
- 密码

**功能点**：
- 登录成功后保存 Token
- 根据 userType 跳转不同首页

---

### 2. 注册界面

**表单字段**：
- 用户名
- 密码 / 确认密码
- 姓名
- 用户类型（学生 / 教师）
- 邮箱
- 手机号

**交互**：
- 密码强度提示
- 必填项校验

**待办事项**：

## 五、证明申请界面（学生）

### 页面结构

```text
┌──────────────────────────────┐
│ 选择证明类型（下拉）        │
├──────────────────────────────┤
│ 申请理由（文本域）          │
├──────────────────────────────┤
│ 动态表单（根据证明类型）    │
├──────────────────────────────┤
│ 份数  是否加急              │
├──────────────────────────────┤
│ [ 提交申请 ]                │
└──────────────────────────────┘
```

### 说明
- 动态表单根据 `certificateType` 渲染
- 申请数据 JSON 自动组装
- 提交前二次确认
